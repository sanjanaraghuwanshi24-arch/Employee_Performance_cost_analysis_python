#Employee_Performace_cost_analysis_pyhthon

## Overview
This project demonstrates data cleaning, transformation, and analysis using Pandas and NumPy.

## Tasks Performed
- Created DataFrames and CSV files
- Filled missing values using running average
- Split names into First Name and Last Name
- Merged multiple DataFrames
- Added bonus calculations
- Implemented promotion and demotion logic
- Created Total Project Cost DataFrame
- Filtered employee records based on city names

## Tools Used
- Python
- Pandas
- NumPy
- Jupyter Notebook
